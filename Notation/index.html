🎯 Goal

Read the note on the staff and pick the correct answer before the clock runs out. Each level gives you 10 questions and 60 seconds.

🎚 Difficulty
  • 🟢 Easy — narrow range, mostly naturals, mostly 3 choices, up to 2 wrong allowed
  • 🟡 Normal — wider range, some accidentals, mostly 4 choices, up to 1 wrong
  • 🔴 Hard — full range with ♯/♭, mostly 5 choices, 0 wrong allowed
  • ⚡ Advanced (unlocked after mastering Hard) — 5 levels, one shared 60s timer
  • 🧘 Practice — endless, no timer, no fail. Pure drill mode.
🎹 Answer modes
  • Letter names — type the note name (C, D, F♯, B♭…)
  • Fingerings — valves for brass (0 1+2 …), slide positions for trombone (1st–7th), simplified keys for woodwinds.
🔁 Wrong answer? Review system

The correct answer flashes for 1s and the missed note is added to the end of the queue. Miss the same note on review and you wait 3s, get a fresh random question, then face the same note again. The queue caps at 20 — overflow means level failed.

🏆 Mastery & progression

Clear all 10 levels of a difficulty to master it. Easy L5 = Normal L1 = Hard L1 in difficulty, so progression overlaps cleanly. Master Hard to unlock ⚡ Advanced.
